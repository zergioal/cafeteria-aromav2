Objetivos de esta guia

  • Entender por que existian problemas de layout antes de Flexbox y Grid

1. El problema historico del layout en CSS

Antes de que existieran Flexbox y Grid, organizar elementos en una pagina web era una pesadilla. Los desarrolladores tenian que usar tecnicas complicadas y poco intuitivas para lograr layouts sencillos.

Lo que se usaba antes (y por que era malo)

Floats y clear: La propiedad float fue creada originalmente para que el texto rodeara imagenes, como en un periodico. Pero los desarrolladores la usaron para crear columnas. El problema era que los elementos flotados se "salian" del flujo normal y habia que usar trucos como clearfix para que el contenedor los reconociera.

/* Esto era el "clearfix hack" - nadie queria escribir esto */
.contenedor::after {
    content: "";
    display: table;
    clear: both;
}

.columna-izquierda {
    float: left;
    width: 50%;
}

.columna-derecha {
    float: right;
    width: 50%;
}

Tablas HTML para layouts: Mucha gente usaba etiquetas <table>, <tr> y <td> para crear layouts. Esto era semanticamente incorrecto (las tablas son para datos, no para diseño), dificulta el SEO y es imposible de mantener.

Centrar verticalmente era casi imposible: Centrar algo vertical y horizontalmente era uno de los problemas mas famosos de CSS. Habia decenas de soluciones, todas con limitaciones. Hoy, con Flexbox, se hace en 2 lineas.

Buenas noticias: Hoy tenemos dos herramientas modernas y poderosas: Flexbox (para layouts de una dimension) y Grid (para layouts de dos dimensiones). Nunca mas necesitaras floats para hacer columnas.

2. Flexbox: Layouts de una dimension

Analogia: Flexbox es como una fila de personas haciendo cola. Puedes decirles que se alineen al centro, que se separen uniformemente, que llenen todo el espacio disponible, o que formen una columna. Todos estan en una linea, ya sea horizontal o vertical.

Flexbox (Flexible Box Layout) resuelve el problema de distribuir espacio y alinear elementos en una sola dimension a la vez: ya sea una fila o una columna.

Como activar Flexbox

Para usar Flexbox solo necesitas escribir display: flex en el elemento padre (llamado "flex container"). Los hijos directos se convierten automaticamente en "flex items".

.contenedor {
    display: flex;  /* Activa Flexbox en este elemento */
    /* Los hijos directos de .contenedor son ahora flex items */
}
/* HTML de ejemplo */
<div class="contenedor">      <!-- flex container -->
    <div class="item">A</div>  <!-- flex item -->
    <div class="item">B</div>  <!-- flex item -->
    <div class="item">C</div>  <!-- flex item -->
</div>

flex-direction: la direccion del eje principal

Por defecto, Flexbox acomoda los elementos en fila (de izquierda a derecha). Puedes cambiar esto con flex-direction.

.contenedor {
    display: flex;
    flex-direction: row;            /* Por defecto: izquierda a derecha */
    flex-direction: row-reverse;    /* Derecha a izquierda */
    flex-direction: column;         /* Arriba a abajo */
    flex-direction: column-reverse; /* Abajo a arriba */
}
/* Visualizacion de flex-direction: */

row (por defecto):
┌─────────────────────────────────┐
│  [A]  [B]  [C]                  │
└─────────────────────────────────┘

row-reverse:
┌─────────────────────────────────┐
│                  [C]  [B]  [A]  │
└─────────────────────────────────┘

column:
┌────────┐
│  [A]   │
│  [B]   │
│  [C]   │
└────────┘

column-reverse:
┌────────┐
│  [C]   │
│  [B]   │
│  [A]   │
└────────┘

justify-content: alinear en el eje principal

justify-content controla como se distribuyen los elementos a lo largo del eje principal (horizontal cuando flex-direction: row).

/* Visualizacion de justify-content (con flex-direction: row): */

flex-start:
┌─────────────────────────────────────┐
│ [A][B][C]                           │
└─────────────────────────────────────┘

flex-end:
┌─────────────────────────────────────┐
│                           [A][B][C] │
└─────────────────────────────────────┘

center:
┌─────────────────────────────────────┐
│              [A][B][C]              │
└─────────────────────────────────────┘

space-between:
┌─────────────────────────────────────┐
│ [A]              [B]              [C]│
└─────────────────────────────────────┘

space-around:
┌─────────────────────────────────────┐
│  ..[A]..      ..[B]..      ..[C]..  │
└─────────────────────────────────────┘

space-evenly:
┌─────────────────────────────────────┐
│   ...[A].....[B].....[C]...         │
└─────────────────────────────────────┘
.contenedor {
    display: flex;
    justify-content: flex-start;    /* Por defecto */
    justify-content: flex-end;      /* Al final */
    justify-content: center;        /* Centrado */
    justify-content: space-between; /* Espacio entre elementos */
    justify-content: space-around;  /* Espacio alrededor de cada elemento */
    justify-content: space-evenly;  /* Espacio igual entre todos */
}

align-items: alinear en el eje secundario

align-items controla la alineacion en el eje perpendicular al principal (vertical cuando flex-direction: row).

.contenedor {
    display: flex;
    height: 200px;             /* Necesitamos altura para ver el efecto */
    align-items: stretch;      /* Por defecto: estira los hijos al 100% */
    align-items: flex-start;   /* Los pega arriba */
    align-items: center;       /* Los centra verticalmente */
    align-items: flex-end;     /* Los pega abajo */
    align-items: baseline;     /* Alinea por la linea base del texto */
}

El truco mas util de Flexbox: centrar perfectamente

Centrar un elemento tanto horizontal como verticalmente es ahora trivial con Flexbox. Esto era increiblemente dificil antes.

/* El truco mas util de Flexbox - centra perfectamente */
.contenedor {
    display: flex;
    justify-content: center;  /* Centrar horizontalmente */
    align-items: center;      /* Centrar verticalmente */
    min-height: 100vh;        /* Ocupa toda la pantalla */
}

/* Resultado:
┌─────────────────────────────────────┐
│                                     │
│                                     │
│           [Elemento]                │
│                                     │
│                                     │
└─────────────────────────────────────┘
*/
Dato: Antes de Flexbox, la busqueda "how to center a div" era una de las mas populares en Stack Overflow. Con display: flex; justify-content: center; align-items: center; el problema desaparecio.

gap: espacio entre elementos

La propiedad gap agrega espacio entre los flex items sin afectar los bordes del contenedor. Es mucho mas limpio que usar margin en cada hijo.

.contenedor {
    display: flex;
    gap: 16px;          /* Espacio de 16px entre todos los elementos */
    gap: 12px 24px;     /* gap-vertical gap-horizontal */
}

/* SIN gap (usando margin - tedioso):
.item + .item { margin-left: 16px; }
/* El ultimo item no tiene espacio a la derecha ✓
/* Pero el primero tampoco... a veces se complica */

/* CON gap (limpio y simple):
.contenedor { gap: 16px; }
/* Gap NO agrega espacio en los bordes del contenedor ✓ */

flex-wrap: que pasa cuando no caben

Por defecto, Flexbox intenta meter todos los elementos en una sola linea. Si no caben, los encoge. Con flex-wrap: wrap los elementos se "bajan" a la siguiente linea.

.contenedor {
    display: flex;
    flex-wrap: nowrap;   /* Por defecto: no ajustar (pueden encogerse) */
    flex-wrap: wrap;     /* Se bajan a la siguiente linea si no caben */
}

/* Con flex-wrap: wrap y elementos de 200px cada uno:

Pantalla ancha:
┌─────────────────────────────────────────────┐
│ [Item 1] [Item 2] [Item 3] [Item 4] [Item 5]│
└─────────────────────────────────────────────┘

Pantalla angosta:
┌──────────────────────┐
│ [Item 1] [Item 2]    │
│ [Item 3] [Item 4]    │
│ [Item 5]             │
└──────────────────────┘
*/

flex: la propiedad de los hijos

La propiedad flex se aplica en los hijos del flex container para controlar cuanto espacio ocupa cada uno.

/* En los HIJOS del flex container */
.item {
    flex: 1;  /* Ocupa una parte del espacio disponible */
}

.item-grande {
    flex: 2;  /* Ocupa el DOBLE que un flex: 1 */
}

/* Ejemplo visual con 3 items:
┌────────────────────────────────────────┐
│  flex:1  │  flex:1  │  flex:1          │
│ (1/3)    │ (1/3)    │ (1/3)            │
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│  flex:1  │    flex:2    │  flex:1      │
│ (1/4)    │   (2/4)      │ (1/4)        │
└────────────────────────────────────────┘
*/

/* flex es un shorthand de: */
.item {
    flex-grow: 1;    /* Cuanto puede crecer */
    flex-shrink: 1;  /* Cuanto puede encogerse */
    flex-basis: 0%;  /* Tamanio base */
}

/* Para la mayoria de casos, flex: 1 es suficiente */

3. Flexbox en Cafeteria Aroma

Conexion con el proyecto real

Flexbox se usa extensamente en Cafeteria Aroma. Veamos los lugares mas importantes donde aparece en el codigo real del proyecto.

La Navbar de Aroma

La barra de navegacion es el ejemplo clasico de Flexbox: el logo a la izquierda y los links a la derecha, todos alineados verticalmente al centro.

/* Navbar principal */
.navbar {
    display: flex;
    align-items: center;            /* Todos al centro vertical */
    justify-content: space-between; /* Logo a la izq, links a la der */
    padding: 12px 24px;
    background: #4A3426;
}

/* Resultado visual:
┌──────────────────────────────────────────────────────────┐
│  [Aroma]          [Inicio][Menu][Carrito][Contacto] [Juan]│
└──────────────────────────────────────────────────────────┘
*/

Los links del nav

/* Los links de navegacion en linea */
.nav-links {
    display: flex;
    list-style: none;     /* Quita los puntos de lista */
    gap: 8px;             /* Espacio entre cada link */
    align-items: center;  /* Alineados al centro vertical */
    margin: 0;
    padding: 0;
}

La cuadricula de productos del menu

Las tarjetas de producto usan flex-wrap: wrap para ajustarse automaticamente al tamanio de la pantalla.

/* Contenedor de tarjetas de productos */
.product-grid {
    display: flex;
    flex-wrap: wrap;  /* Se ajustan a la siguiente linea si no caben */
    gap: 20px;
}

.product-card {
    flex: 1;
    min-width: 240px;  /* Minimo 240px de ancho */
    max-width: 320px;  /* Maximo 320px de ancho */
    /* Con min y max-width, las tarjetas son flexibles pero con limites */
}

La estructura interna de cada tarjeta

Dentro de cada tarjeta tambien se usa Flexbox para organizar imagen, informacion, precio y boton.

/* La tarjeta es una columna */
.product-card {
    display: flex;
    flex-direction: column;  /* Imagen arriba, info abajo */
    border-radius: 12px;
    overflow: hidden;
    background: white;
    box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

/* La seccion de informacion ocupa todo el espacio disponible */
.product-info {
    display: flex;
    flex-direction: column;
    flex: 1;               /* Ocupa todo el espacio disponible */
    padding: 16px;
}

/* El footer de la tarjeta: precio + boton */
.product-footer {
    display: flex;
    justify-content: space-between;  /* Precio izq, boton der */
    align-items: center;             /* Centrado vertical */
    margin-top: auto;                /* Empuja al fondo de la tarjeta */
    padding-top: 12px;
}

/* Resultado visual de la tarjeta:
┌──────────────────┐
│  [Imagen cafe]   │  <- product-image
├──────────────────┤
│ Cafe Americano   │  <- product-info
│ Descripcion...   │    (flex: 1, ocupa todo)
│                  │
│ Bs.12 [Agregar]  │  <- product-footer (margin-top: auto)
└──────────────────┘
*/
Truco de Flexbox: Usar margin-top: auto en un flex item lo empuja hasta el fondo del contenedor. Es perfecto para tarjetas donde quieres que el precio y boton siempre esten al fondo, sin importar cuanto texto tenga la descripcion.

4. CSS Grid: Layouts de dos dimensiones

Analogia: Si Flexbox es una fila de personas en una cola (una dimension), Grid es un tablero de ajedrez (dos dimensiones). Defines filas Y columnas al mismo tiempo, y puedes colocar elementos en cualquier celda, o hacer que ocupen varias celdas a la vez.

CSS Grid es el sistema de layout mas poderoso de CSS. Permite crear layouts complejos de dos dimensiones con un codigo sorprendentemente limpio.

Como activar Grid

.grid-contenedor {
    display: grid;                          /* Activa Grid */
    grid-template-columns: 200px 200px 200px; /* 3 columnas de 200px */
    grid-template-rows: 100px 100px;          /* 2 filas de 100px */
    gap: 16px;                              /* Espacio entre celdas */
}

/* Resultado visual:
┌──────────┬──────────┬──────────┐
│ 200px    │ 200px    │ 200px    │ 100px
├──────────┼──────────┼──────────┤
│          │          │          │ 100px
└──────────┴──────────┴──────────┘
*/

La unidad fr (fraccion disponible)

La unidad fr (fraccion) es especial de Grid. Representa una fraccion del espacio disponible en el contenedor. Es como repartir una pizza.

.grid {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    /* El espacio disponible se divide en 4 partes (1+2+1):
       - Columna 1: 1/4 del espacio (25%)
       - Columna 2: 2/4 del espacio (50%) <- la del medio es el doble
       - Columna 3: 1/4 del espacio (25%)
    */
}

/* Resultado visual:
┌────────┬────────────────┬────────┐
│  1fr   │      2fr       │  1fr   │
│  (25%) │     (50%)      │  (25%) │
└────────┴────────────────┴────────┘
*/

repeat(): para no repetir codigo

La funcion repeat() evita escribir lo mismo varias veces.

.grid {
    display: grid;

    /* Estas dos lineas hacen lo mismo: */
    grid-template-columns: 1fr 1fr 1fr 1fr;     /* 4 columnas iguales */
    grid-template-columns: repeat(4, 1fr);       /* Mas limpio */

    /* Tambien con px: */
    grid-template-columns: 200px 200px 200px;   /* Repetitivo */
    grid-template-columns: repeat(3, 200px);    /* Limpio */
}

auto-fit y minmax: el patron mas util de Grid

Esta combinacion es magica. Permite que la grilla se ajuste automaticamente a cualquier tamanio de pantalla, sin necesitar media queries.

.product-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}

/*
Desglosando:
- repeat(...)         -> repetir columnas
- auto-fit            -> crea tantas columnas como quepan en el espacio
- minmax(250px, 1fr)  -> cada columna tiene minimo 250px y maximo 1fr

Comportamiento segun ancho de pantalla:
- 300px  de ancho: 1 columna  (solo cabe 1 de minimo 250px)
- 600px  de ancho: 2 columnas (caben 2 de 250px cada una)
- 900px  de ancho: 3 columnas (caben 3 de 250px cada una)
- 1200px de ancho: 4 columnas (caben 4 de 250px cada una)

Resultado en desktop (1200px):
┌──────────┬──────────┬──────────┬──────────┐
│ Card 1   │ Card 2   │ Card 3   │ Card 4   │
├──────────┼──────────┼──────────┼──────────┤
│ Card 5   │ Card 6   │ Card 7   │ Card 8   │
└──────────┴──────────┴──────────┴──────────┘

Resultado en movil (350px):
┌──────────────────────┐
│       Card 1         │
├──────────────────────┤
│       Card 2         │
├──────────────────────┤
│       Card 3         │
└──────────────────────┘
*/
Patron clave: repeat(auto-fit, minmax(X, 1fr)) es uno de los patrones mas importantes de CSS moderno. Con una sola linea de CSS creas una grilla completamente responsive. Aprendelo de memoria.

Colocar elementos en posiciones especificas

Con Grid puedes controlar exactamente en que columna y fila va cada elemento, o hacer que ocupe multiples celdas.

/* Grid de 3 columnas */
.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 16px;
}

/* Un elemento que ocupa 2 columnas */
.item-destacado {
    grid-column: 1 / 3;  /* Desde linea de columna 1 hasta linea 3 */
    grid-row: 1 / 2;     /* Primera fila */
}

/* Un elemento que ocupa TODAS las columnas */
.item-full-width {
    grid-column: 1 / -1; /* Desde la primera hasta la ULTIMA linea */
    /* -1 siempre es la ultima linea, sin importar cuantas columnas haya */
}

/* Visualizacion:
┌──────────────────────┬──────────┐
│ item-destacado       │ item 2   │
│ (ocupa col 1 y 2)    │          │
├──────────┬───────────┴──────────┤
│ item 3   │ item 4               │
├──────────┴──────────────────────┤
│         item-full-width         │
│ (ocupa las 3 columnas)          │
└─────────────────────────────────┘
*/

Grid en el panel Admin de Aroma

/* Estadisticas del panel de administracion */
.admin-stats {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 20px;
    /* Se ajusta automaticamente: en movil 1 col, en desktop 4 cols */
}

/* Layout principal del panel admin: sidebar + contenido */
.admin-layout {
    display: grid;
    grid-template-columns: 250px 1fr;  /* Sidebar fijo + contenido flexible */

}

/* Resultado:
┌──────────────────────────────────────────────┐
│               Header (auto)                  │ <- row 1
├───────────────┬──────────────────────────────┤
│ Sidebar       │ Contenido principal          │ <- row 2 (1fr)
│ (250px fijo)  │ (1fr, ocupa todo el resto)   │
│               │                              │
│               │                              │
└───────────────┴──────────────────────────────┘
*/

5. Flexbox vs Grid: ¿Cuando usar cada uno?

Esta es la pregunta que todos los principiantes hacen. La respuesta corta: Flexbox para una dimension, Grid para dos dimensiones. Pero hay mas matices.

Situacion Usar Por que
Navbar con logo y links Flexbox Es una linea horizontal, una sola dimension
Formulario con campos Flexbox column Cada campo es una fila, una sola dimension
Galeria de productos Grid Necesita filas Y columnas, dos dimensiones
Contenido interno de una tarjeta Flexbox El contenido es lineal (columna)
Dashboard con estadisticas Grid Layout 2D complejo
Centrar un elemento Flexbox justify-content: center; align-items: center
Lista de botones en linea Flexbox Estan en una sola linea
Layout de pagina completo Grid Header, sidebar, contenido, footer en 2D
Barra de busqueda con icono Flexbox Input y boton en una sola linea
Regla practica: Si el contenido fluye en una sola linea (ya sea horizontal o vertical), usa Flexbox. Si necesitas controlar filas Y columnas simultaneamente, usa Grid. Y recuerda: puedes combinarlos, un elemento puede ser un Grid container cuyos hijos son Flex containers.

Resumen de la Guia 04

  • Flexbox organiza elementos en una dimension: fila (row) o columna (column)
  • justify-content controla la alineacion en el eje principal (horizontal en row)
  • align-items controla la alineacion en el eje secundario (vertical en row)
  • Para centrar algo: display: flex; justify-content: center; align-items: center;
  • gap agrega espacio entre elementos sin afectar los bordes del contenedor
  • flex-wrap: wrap permite que los elementos bajen a la siguiente linea
  • flex: 1 en los hijos hace que ocupen el espacio disponible en partes iguales
  • CSS Grid organiza elementos en dos dimensiones: filas y columnas
  • La unidad fr representa una fraccion del espacio disponible
  • repeat(auto-fit, minmax(X, 1fr)) crea una grilla responsive sin media queries
  • Usa Flexbox para una dimension, Grid para dos dimensiones
  • En Aroma: navbar y tarjetas internas usan Flexbox, galeria de productos y layouts de pagina usan Grid
