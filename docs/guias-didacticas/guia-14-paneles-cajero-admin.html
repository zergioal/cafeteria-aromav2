Guía 14 – Paneles de Cajero y Administrador

Módulo 6: Integración | Gestión del negocio en tiempo real

Nivel: Avanzado Tiempo estimado: 6 horas Prerequisito: Guía 13

Objetivos de esta guía

  • Construir el panel del cajero con gestión de pedidos en tiempo real
  • Implementar el cambio de estados del pedido con feedback visual
  • Desarrollar el panel del administrador con dashboard de estadísticas
  • Crear el formulario de gestión de productos (CRUD completo)
  • Construir el endpoint de dashboard en el backend
  • Implementar auto-refresh para mantener los datos actualizados

1. Roles y permisos en Aroma

La Cafetería Aroma tiene tres tipos de usuarios, cada uno con su propio nivel de acceso:

Cliente (client)

  • Ver el menú público
  • Agregar productos al carrito
  • Hacer pedidos
  • Ver sus propios pedidos

Cajero (cashier)

  • Ver todos los pedidos activos
  • Cambiar el estado de los pedidos
  • Verificar métodos de pago
  • Filtrar pedidos por fecha

Administrador (admin)

  • Todo lo del cajero
  • Ver estadísticas del negocio
  • Crear/editar/eliminar productos
  • Gestionar categorías
  • Ver ventas históricas

Público (sin login)

  • Ver el menú
  • Ver la descripción de productos
  • Ver información de contacto
  • Agregar al carrito (sin confirmar)

2. El Panel del Cajero — Frontend

Analogía: El panel del cajero es como la pantalla de comandas en la cocina de un restaurante: muestra todos los pedidos activos, ordenados por tiempo, y el cajero puede actualizar el estado conforme avanza el proceso. Sin esta pantalla, el cajero no sabría qué pedidos tiene pendientes.

¿Qué puede hacer el cajero?

  • Ver pedidos nuevos en tiempo casi real (auto-refresh cada 30 segundos)
  • Cambiar el estado: Pendiente → Confirmado → Preparando → Listo → Entregado
  • Ver el detalle completo de cada pedido (items, dirección, teléfono)
  • Filtrar pedidos por estado o fecha
  • Cancelar pedidos con motivo
// frontend/js/pages/cashier.js
const CashierPage = {
    orders: [],
    selectedOrderId: null,
    refreshInterval: null,

    async init() {
        // Verificar que el usuario tiene permiso
        const session = await Auth.getSession();
        const role = session?.user?.user_metadata?.role;

        if (role !== 'cashier' && role !== 'admin') {
            mostrarToast('Acceso denegado', 'error');
            App.showTab('home');
            return;
        }

        await this.cargarPedidos();
        this.setupFiltros();
        this.setupAcciones();
        this.setupAutoRefresh();
    },

    async cargarPedidos(filtros = {}) {
        try {
            // Construir los query params según los filtros seleccionados
            const params = new URLSearchParams();
            if (filtros.status) params.set('status', filtros.status);
            if (filtros.fecha) params.set('fecha', filtros.fecha);

            const queryString = params.toString();
            const url = `/api/cashier/orders${queryString ? '?' + queryString : ''}`;

            const { orders } = await Api.get(url);
            this.orders = orders;
            this.renderizarListaPedidos();
            this.actualizarContadores();

        } catch (error) {
            mostrarToast('Error cargando pedidos', 'error');
            console.error(error);
        }
    },

    renderizarListaPedidos() {
        const lista = document.getElementById('ordersList');

        if (this.orders.length === 0) {
            lista.innerHTML = `
                <div class="no-orders">
                    <p>No hay pedidos que mostrar</p>
                </div>
            `;
            return;
        }

        lista.innerHTML = this.orders.map(order => `
            <div class="order-card status-${order.status}"
                 data-id="${order.id}">
                <div class="order-card-header">
                    <strong class="order-code">${order.code}</strong>
                    <span class="status-badge badge-${order.status}">
                        ${this.getStatusLabel(order.status)}
                    </span>
                    <span class="order-time">${this.formatTime(order.created_at)}</span>
                </div>
                <div class="order-customer">
                    <span>Cliente: ${escHtml(order.customer_name)}</span>
                    <span>Tel: ${order.customer_phone}</span>
                </div>
                <div class="order-items-preview">
                    ${order.items?.map(i =>
                        `<span class="item-tag">${i.quantity}x ${escHtml(i.product_name)}</span>`
                    ).join('') || ''}
                </div>
                <div class="order-footer">
                    <span class="order-total">Bs. ${order.total.toFixed(2)}</span>
                    <span class="payment-method">${order.payment_method?.toUpperCase()}</span>
                    <div class="order-actions">
                        ${this.getActionButtons(order)}
                    </div>
                </div>
            </div>
        `).join('');

        // Event delegation para los botones de acción
        lista.addEventListener('click', async (e) => {
            const btn = e.target.closest('[data-action]');
            if (!btn) return;

            const orderId = btn.closest('[data-id]')?.dataset.id;
            const action = btn.dataset.action;

            if (action && orderId) {
                await this.cambiarEstado(orderId, action);
            }
        });
    },

    // Determinar qué botones mostrar según el estado actual
    getActionButtons(order) {
        const nextStates = {
            pending: [{ label: 'Confirmar', next: 'confirmed', style: 'primary' }],
            confirmed: [{ label: 'Preparando', next: 'preparing', style: 'warning' }],
            preparing: [{ label: 'Listo', next: 'ready', style: 'success' }],
            ready: [{ label: 'Entregado', next: 'delivered', style: 'success' }],
            delivered: [],
            cancelled: []
        };

        const buttons = nextStates[order.status] || [];
        const cancelBtn = ['pending', 'confirmed'].includes(order.status)
            ? `<button class="btn-action btn-cancel" data-action="cancelled">Cancelar</button>`
            : '';

        return buttons.map(b =>
            `<button class="btn-action btn-${b.style}" data-action="${b.next}">
                ${b.label}
             </button>`
        ).join('') + cancelBtn;
    },

    async cambiarEstado(orderId, nuevoEstado) {
        try {
            const { order } = await Api.patch(
                `/api/cashier/orders/${orderId}/status`,
                { status: nuevoEstado }
            );
            mostrarToast(
                `Pedido ${order.code}: ${this.getStatusLabel(nuevoEstado)}`,
                'success'
            );
            await this.cargarPedidos(); // Refrescar la lista
        } catch (error) {
            mostrarToast('Error al actualizar el pedido', 'error');
        }
    },

    getStatusLabel(status) {
        const labels = {
            pending: 'Pendiente',
            confirmed: 'Confirmado',
            preparing: 'Preparando',
            ready: 'Listo',
            delivered: 'Entregado',
            cancelled: 'Cancelado'
        };
        return labels[status] || status;
    },

    formatTime(isoDate) {
        const date = new Date(isoDate);
        return date.toLocaleTimeString('es-BO', {
            hour: '2-digit',
            minute: '2-digit'
        });
    },

    actualizarContadores() {
        const pendientes = this.orders.filter(o => o.status === 'pending').length;
        const enProceso = this.orders.filter(o =>
            ['confirmed', 'preparing'].includes(o.status)
        ).length;

        const elPendientes = document.getElementById('countPending');
        const elProceso = document.getElementById('countInProgress');
        if (elPendientes) elPendientes.textContent = pendientes;
        if (elProceso) elProceso.textContent = enProceso;
    },

    setupFiltros() {
        document.getElementById('filterStatus')?.addEventListener('change', (e) => {
            this.cargarPedidos({ status: e.target.value || undefined });
        });

        document.getElementById('filterFecha')?.addEventListener('change', (e) => {
            this.cargarPedidos({ fecha: e.target.value || undefined });
        });

        document.getElementById('btnRefresh')?.addEventListener('click', () => {
            this.cargarPedidos();
        });
    },

    setupAutoRefresh() {
        // Refrescar automáticamente cada 30 segundos
        this.refreshInterval = setInterval(() => {
            this.cargarPedidos();
        }, 30000);

        // Limpiar el interval cuando el usuario cambia de tab
        document.querySelectorAll('[data-tab]').forEach(el => {
            el.addEventListener('click', () => {
                if (el.dataset.tab !== 'cashier' && this.refreshInterval) {
                    clearInterval(this.refreshInterval);
                    this.refreshInterval = null;
                }
            });
        });
    },

    setupAcciones() { /* configuración adicional de UI */ }
};
Auto-refresh con setInterval: setInterval(fn, 30000) ejecuta la función cada 30 segundos. Es importante guardar el ID que devuelve y limpiarlo con clearInterval(id) cuando no se necesita más, para evitar llamadas innecesarias al servidor cuando el usuario está en otra sección.

3. El Panel del Administrador — Frontend

El administrador tiene acceso a una vista más completa del negocio: estadísticas, gestión de todo el catálogo de productos, y reportes.

// frontend/js/pages/admin.js
const AdminPage = {
    productos: [],
    categorias: [],
    editandoId: null,

    async init() {
        // Cargar todo en paralelo para mayor rapidez
        await Promise.all([
            this.cargarEstadisticas(),
            this.cargarProductos(),
            this.cargarCategorias()
        ]);
        this.setupFormularioProducto();
        this.setupTabs();
    },

    // ===== DASHBOARD =====
    async cargarEstadisticas() {
        try {
            const { stats } = await Api.get('/api/admin/dashboard');

            // Actualizar las tarjetas del dashboard
            this.actualizarCard('totalVentas', `Bs. ${stats.totalVentasHoy.toFixed(2)}`);
            this.actualizarCard('totalPedidos', stats.totalPedidosHoy);
            this.actualizarCard('pedidosPendientes', stats.pedidosPendientes);
            this.actualizarCard('productosActivos', stats.productosActivos);

        } catch (error) {
            console.error('Error cargando estadísticas:', error);
        }
    },

    actualizarCard(id, valor) {
        const el = document.getElementById(id);
        if (el) el.textContent = valor;
    },

    // ===== GESTIÓN DE PRODUCTOS =====
    async cargarProductos() {
        try {
            const { products } = await Api.get('/api/products?all=true');
            this.productos = products;
            this.renderizarTablaProductos();
        } catch (error) {
            mostrarToast('Error cargando productos', 'error');
        }
    },

    renderizarTablaProductos() {
        const tbody = document.getElementById('productosTableBody');
        if (!tbody) return;

        tbody.innerHTML = this.productos.map(p => `
            <tr class="${p.activo ? '' : 'inactivo'}">
                <td>
                    ${p.imagen_url
                        ? `<img src="${p.imagen_url}" alt="${escHtml(p.nombre)}" width="50" height="50" style="border-radius:6px;object-fit:cover">`
                        : '—'
                    }
                </td>
                <td>
                    <strong>${escHtml(p.nombre)}</strong>
                    <br>
                    <small style="color:#888">${escHtml(p.descripcion || '')}...</small>
                </td>
                <td>${escHtml(p.category?.nombre || '—')}</td>
                <td>Bs. ${p.precio.toFixed(2)}</td>
                <td>
                    <span class="status-chip ${p.activo ? 'activo' : 'inactivo'}">
                        ${p.activo ? 'Activo' : 'Inactivo'}
                    </span>
                </td>
                <td>
                    <button class="btn-icon" onclick="AdminPage.editarProducto('${p.id}')"
                            title="Editar">Editar</button>
                    <button class="btn-icon btn-danger"
                            onclick="AdminPage.toggleActivo('${p.id}', ${p.activo})"
                            title="${p.activo ? 'Desactivar' : 'Activar'}">
                        ${p.activo ? 'Desactivar' : 'Activar'}
                    </button>
                </td>
            </tr>
        `).join('');
    },

    editarProducto(id) {
        const producto = this.productos.find(p => p.id === id);
        if (!producto) return;

        this.editandoId = id;

        // Llenar el formulario con los datos del producto
        document.getElementById('prodNombre').value = producto.nombre;
        document.getElementById('prodDescripcion').value = producto.descripcion || '';
        document.getElementById('prodPrecio').value = producto.precio;
        document.getElementById('prodCosto').value = producto.costo || '';
        document.getElementById('prodCategoria').value = producto.category_id;
        document.getElementById('prodImagen').value = producto.imagen_url || '';

        // Cambiar el título del formulario
        document.getElementById('formTitle').textContent = 'Editar Producto';
        document.getElementById('btnFormSubmit').textContent = 'Guardar Cambios';

        // Hacer scroll al formulario
        document.getElementById('productoForm').scrollIntoView({ behavior: 'smooth' });
    },

    async guardarProducto(datos) {
        try {
            if (this.editandoId) {
                await Api.put(`/api/admin/products/${this.editandoId}`, datos);
                mostrarToast('Producto actualizado correctamente', 'success');
            } else {
                await Api.post('/api/admin/products', datos);
                mostrarToast('Producto creado correctamente', 'success');
            }

            this.editandoId = null;
            this.resetFormulario();
            await this.cargarProductos();

        } catch (error) {
            mostrarToast(error.message || 'Error al guardar producto', 'error');
        }
    },

    async toggleActivo(id, estadoActual) {
        const accion = estadoActual ? 'desactivar' : 'activar';
        if (!confirm(`¿Deseas ${accion} este producto?`)) return;

        try {
            await Api.patch(`/api/admin/products/${id}`, { activo: !estadoActual });
            mostrarToast(`Producto ${accion === 'activar' ? 'activado' : 'desactivado'}`, 'success');
            await this.cargarProductos();
        } catch (error) {
            mostrarToast('Error al cambiar el estado del producto', 'error');
        }
    },

    setupFormularioProducto() {
        const form = document.getElementById('productoForm');
        if (!form) return;

        form.addEventListener('submit', async (e) => {
            e.preventDefault();

            const datos = {
                nombre: document.getElementById('prodNombre').value.trim(),
                descripcion: document.getElementById('prodDescripcion').value.trim(),
                precio: parseFloat(document.getElementById('prodPrecio').value),
                costo: parseFloat(document.getElementById('prodCosto').value) || null,
                category_id: document.getElementById('prodCategoria').value,
                imagen_url: document.getElementById('prodImagen').value.trim() || null
            };

            // Validación básica
            if (!datos.nombre || !datos.precio || !datos.category_id) {
                mostrarToast('Nombre, precio y categoría son requeridos', 'warning');
                return;
            }

            if (datos.precio <= 0) {
                mostrarToast('El precio debe ser mayor a 0', 'warning');
                return;
            }

            const btn = form.querySelector('button[type="submit"]');
            btn.disabled = true;
            await this.guardarProducto(datos);
            btn.disabled = false;
        });

        // Botón "Nuevo producto"
        document.getElementById('btnNuevoProducto')?.addEventListener('click', () => {
            this.editandoId = null;
            this.resetFormulario();
        });
    },

    resetFormulario() {
        document.getElementById('productoForm')?.reset();
        document.getElementById('formTitle').textContent = 'Nuevo Producto';
        document.getElementById('btnFormSubmit').textContent = 'Crear Producto';
    },

    async cargarCategorias() {
        try {
            const { categories } = await Api.get('/api/categories');
            this.categorias = categories;

            // Llenar el select de categorías en el formulario
            const select = document.getElementById('prodCategoria');
            if (select) {
                select.innerHTML = `<option value="">Seleccionar categoría...</option>` +
                    categories.map(c =>
                        `<option value="${c.id}">${escHtml(c.nombre)}</option>`
                    ).join('');
            }
        } catch (error) {
            console.error('Error cargando categorías:', error);
        }
    },

    setupTabs() {
        // Navegación entre secciones del panel admin
        document.querySelectorAll('.admin-tab-btn').forEach(btn => {
            btn.addEventListener('click', () => {
                const section = btn.dataset.section;

                document.querySelectorAll('.admin-section').forEach(s =>
                    s.style.display = 'none'
                );
                document.getElementById(`admin-${section}`)?.style.setProperty('display', 'block');

                document.querySelectorAll('.admin-tab-btn').forEach(b =>
                    b.classList.remove('active')
                );
                btn.classList.add('active');
            });
        });
    }
};

4. El Dashboard del Administrador — Backend

El endpoint del dashboard agrega estadísticas del negocio desde múltiples tablas de la base de datos.

// backend/src/controllers/adminController.js

// GET /api/admin/dashboard
export async function getDashboard(req, res) {
    try {
        const hoy = new Date().toISOString().split('T')[0]; // "2024-11-15"
        const inicioHoy = `${hoy}T00:00:00`;
        const finHoy = `${hoy}T23:59:59`;

        // Ejecutar todas las consultas en paralelo para mayor velocidad
        const [
            pedidosHoyResult,
            pendientesResult,
            productosResult,
            ventasSemanaResult
        ] = await Promise.all([
            // Pedidos de hoy (excluir cancelados)
            supabase
                .from('orders')
                .select('id, total, status')
                .gte('created_at', inicioHoy)
                .lte('created_at', finHoy)
                .neq('status', 'cancelled'),

            // Pedidos pendientes ahora mismo
            supabase
                .from('orders')
                .select('id', { count: 'exact' })
                .eq('status', 'pending'),

            // Productos activos
            supabase
                .from('products')
                .select('id', { count: 'exact' })
                .eq('activo', true),

            // Ventas de los últimos 7 días (solo entregados)
            supabase
                .from('orders')
                .select('created_at, total')
                .gte('created_at', (() => {
                    const d = new Date();
                    d.setDate(d.getDate() - 7);
                    return d.toISOString();
                })())
                .eq('status', 'delivered')
        ]);

        const pedidosHoy = pedidosHoyResult.data || [];
        const totalVentasHoy = pedidosHoy.reduce((sum, p) => sum + (p.total || 0), 0);

        // Agrupar ventas por día para el gráfico de los últimos 7 días
        const ventasPorDia = {};
        (ventasSemanaResult.data || []).forEach(order => {
            const dia = order.created_at.split('T')[0];
            ventasPorDia[dia] = (ventasPorDia[dia] || 0) + order.total;
        });

        res.json({
            stats: {
                totalVentasHoy,
                totalPedidosHoy: pedidosHoy.length,
                pedidosPendientes: pendientesResult.count || 0,
                productosActivos: productosResult.count || 0
            },
            ventasPorDia
        });

    } catch (error) {
        console.error('Error en dashboard:', error);
        res.status(500).json({ error: 'Error al obtener estadísticas' });
    }
}
Promise.all para consultas paralelas: En lugar de esperar una consulta y luego la siguiente (lento), Promise.all([]) lanza todas las consultas al mismo tiempo y espera a que todas terminen. Si tienes 4 consultas que tardan 100ms cada una, con Promise.all tarda ~100ms en total en lugar de ~400ms.

6. Seguridad en los paneles de gestión

Más allá de la autenticación en el backend, el frontend también debe ser defensivo para evitar errores y confusión:

Validaciones del lado del cliente

  • Verificar el rol al inicializar: Si el módulo CashierPage.init() detecta que el usuario no tiene el rol correcto, redirige inmediatamente.
  • Escapar HTML en datos del usuario: Siempre usar escHtml() al insertar datos que vienen de la BD en el DOM. Esto previene ataques XSS.
  • Confirmaciones antes de acciones destructivas: Usar confirm() antes de cancelar un pedido o desactivar un producto.
  • Deshabilitar botones durante peticiones: Evitar que el cajero haga doble clic y actualice el estado dos veces.
// La función escHtml previene ataques XSS al escapar caracteres especiales
function escHtml(str) {
    if (typeof str !== 'string') return str;
    return str
        .replace(/&/g, '&amp;')
        .replace(/</g, '&lt;')
        .replace(/>/g, '&gt;')
        .replace(/"/g, '&quot;')
        .replace(/'/g, '&#039;');
}

// Ejemplo de uso seguro
tbody.innerHTML = productos.map(p => `
    <td>${escHtml(p.nombre)}</td>  <!-- Seguro -->
    <td>${p.nombre}</td>            <!-- Inseguro si p.nombre viene de la BD -->
`).join('');
XSS — Cross-Site Scripting: Si un usuario malintencionado registra su nombre como <script>alert('hackeado')</script> y el sistema lo muestra sin escapar, ese script se ejecutaría en el navegador de cualquier cajero que lo vea. Escapar siempre el HTML externo previene este tipo de ataque.

Dashboard: Abrir el panel de administrador. Verificar que las estadísticas del día se muestran (ventas, pedidos, pendientes, productos activos).

Crear un producto: Ir a la sección de productos, llenar el formulario con un nuevo producto y guardarlo. Verificar que aparece en la tabla y en el menú público.
7
Editar y desactivar: Editar el precio del producto recién creado. Luego desactivarlo y verificar que ya no aparece en el menú público.

Resumen de la Guía 14

  • El sistema tiene tres roles: cliente, cajero y admin, con acceso progresivo.
  • El panel del cajero muestra pedidos activos con auto-refresh cada 30 segundos.
  • Los botones de acción del cajero se generan dinámicamente según el estado actual del pedido.
  • El panel del admin incluye un dashboard con estadísticas del día calculadas en el backend.
  • Promise.all lanza múltiples consultas en paralelo para optimizar la velocidad.
  • El CRUD de productos incluye soft delete (desactivar, no borrar) para preservar el historial.
  • La función escHtml() previene ataques XSS al escapar datos dinámicos en el DOM.
  • Los formularios deben validar en el cliente Y en el servidor; ninguna de las dos valida sola es suficiente.
